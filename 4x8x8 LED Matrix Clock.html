4x8x8 LED Matrix Clock

This project displays the current time using four 8×8 LED matrix displays driven by a microcontroller. It’s a visually engaging way to explore display control, real-time clocks, and I2C communication. Below, you can see the bill of materials, the schematic, PCB layout, and 3D model of the project.
