ESP32 Clone Devkit

This custom development board is built around the ESP32 chip and designed for wireless IoT applications. It covers RF design basics and USB-to-serial integration for easy programming. The board includes a microSD card interface for data logging, allowing sensor data to be stored for later analysis. Resources include the bill of materials, schematic, PCB layout, and 3D model.
