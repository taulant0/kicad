MCU Datalogger

This device collects data from sensors and stores it on a microSD card for later analysis. It is designed to help understand how microcontrollers interface with storage devices and manage data logging tasks, making it suitable for applications such as temperature monitoring, motion tracking, or environmental data collection. The project files include the bill of materials, schematic, PCB layout, and 3D model.
Notably, the schematic was organized using hierarchical sheets in KiCad, which helped structure the design modularly and improve clarity.
The project was designed and managed using Git, including one schematic and two PCB layout versions: a 2-layer and a 4-layer design.
