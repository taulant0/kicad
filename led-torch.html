LED Torch Circuit

This is a simple battery-powered torch that lights up an LED when a button is pressed. It’s a great introductory project for understanding how basic components like resistors, LEDs, and switches work together in a functional circuit. Below, you can see the bill of materials, schematic, PCB layout, and 3D model of the project.
